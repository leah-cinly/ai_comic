# 自动化使用说明

## 更新学习资料

在 vault 根目录运行：

```powershell
.\scripts\update-learning-vault.ps1
```

它会根据核心资料源生成一份当天的学习更新笔记。

## 同步到 GitHub

第一次需要先设置远程仓库：

```powershell
git remote add origin https://github.com/你的用户名/你的仓库名.git
```

之后运行：

```powershell
.\scripts\sync-to-github.ps1
```

脚本会提交当天变化并推送到 GitHub。

## Obsidian 平板端学习

平板端如果没有 Obsidian Sync，可以用 GitHub 作为中转：

- 电脑端：运行同步脚本，把 vault 推到 GitHub。
- 平板端：用支持 Git 的方式拉取仓库，或直接在 GitHub App / 浏览器阅读 Markdown。

# 资料源清单

抓取资料不是为了囤链接，而是为了形成三类笔记：

- 理论卡：一个概念 + 一个例子 + 一个练习。
- 工具卡：一个工具能力 + 限制 + 适合的镜头类型。
- 案例卡：一个短片/镜头 + 为什么有效 + 可以迁移的做法。

## 第一阶段优先资料

### 动画理论与书籍

- 《The Illusion of Life: Disney Animation》：12 条动画原则的经典来源。https://books.disney.com/book/the-illusion-of-life/
- 《The Animator's Survival Kit》：时间控制、走路、重量、关键姿势。https://www.routledge.com/The-Animators-Survival-Kit/Williams/p/book/9780571238347
- Pixar in a Box：动画叙事、角色、色彩、模拟等免费课程。https://www.khanacademy.org/computing/pixar

### 分镜与逐帧动画

- Krita Animation Manual：逐帧动画、时间线、洋葱皮、分镜、animatic。https://docs.krita.org/en/user_manual/animation.html
- Storyboarder GitHub：开源分镜工具。https://github.com/wonderunit/storyboarder
- OpenToonz GitHub：开源 2D 动画制作软件。https://github.com/opentoonz/opentoonz

### AI 图像与提示词

- PixAI：https://pixai.art/
- Lexica：https://lexica.art/
- OpenArt：https://openart.ai/

使用方式：主要用于积累风格词、构图词、角色一致性词，不要把平台热门图直接当作审美答案。

### AI 视频

- Runway Academy：AI 视频课程、镜头运动和提示词指南。https://academy.runwayml.com/
- Pika：https://pika.art/
- Luma：https://lumalabs.ai/

注意：OpenAI 官方帮助中心显示 Sora web/app 体验已在 2026-04-26 停止，Sora API 将在 2026-09-24 停止；学习路线里不要把 Sora 当作长期主工具。https://help.openai.com/en/articles/20001152-what-to-know-about-the-sora-discontinuation

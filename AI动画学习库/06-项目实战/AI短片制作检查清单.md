# AI 短片制作检查清单

## 开始前

- [ ] 有一句话故事。
- [ ] 有 5 拍结构。
- [ ] 有 12-20 个镜头清单。
- [ ] 有角色主参考图。
- [ ] 有场景和色彩脚本。
- [ ] 有声音方向。

## 生成前

- [ ] 每个镜头有明确景别和机位。
- [ ] 每个镜头只有一个主要信息。
- [ ] 角色一致性约束写入提示词。
- [ ] 光源方向可连续。
- [ ] 参考图来源已记录。

## 生成中

- [ ] 每个镜头保存提示词。
- [ ] 每个输出保存版本号。
- [ ] 失败结果不要删，记录原因。
- [ ] 同一工具用同一任务做对比。
- [ ] 镜头生成后立刻放进时间线测试。

## 剪辑中

- [ ] 动作能接上。
- [ ] 视线方向不乱。
- [ ] 音效有节奏。
- [ ] 音乐没有盖过重要动作。
- [ ] 字幕不挡主体。

## 发布前

- [ ] 导出横屏/竖屏需要的版本。
- [ ] 封面图清楚。
- [ ] 授权来源记录完整。
- [ ] 项目复盘已写。
- [ ] 最有效提示词已沉淀到 [[提示词素材库]]。

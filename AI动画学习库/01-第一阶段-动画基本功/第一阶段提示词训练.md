# 第一阶段提示词训练

这一页专门训练“从美术描述到镜头控制”的提示词。PixAI、Lexica、OpenArt 可以用来积累风格词、构图词和角色一致性词，但第一阶段不要只收藏漂亮图，要反向拆解它们为什么有效。

素材入口：

- PixAI：https://pixai.art/
- Lexica：https://lexica.art/
- OpenArt：https://openart.ai/
- Runway Academy：https://academy.runwayml.com/

## 观察字段

每看到一张可学习的图，记录：主体、动作、情绪、景别、机位、光线、风格、连续性。

## 基础提示词公式

```text
[subject], [specific action], [shot size], [camera angle], [emotion], [lighting], [background supports story], [animation style], [continuity constraints]
```

中文理解：主体 + 具体动作 + 景别 + 机位 + 情绪 + 光线 + 背景功能 + 动画风格 + 连续性约束。

## 角色一致性模板

```text
same character, consistent facial features, consistent hairstyle, consistent outfit, simple memorable silhouette, animation character design sheet, front view, side view, back view, expression sheet
```

## 镜头提示词模板

```text
cinematic animated short film still, medium shot, clear staging, readable silhouette, character hesitates before making a decision, warm key light from screen left, simple background supports the emotion, consistent character design
```

## 动作提示词模板

```text
pose-to-pose animation keyframe, anticipation pose, clear line of action, natural arc of motion, exaggerated but believable expression, follow-through in hair and clothing
```

## 负面提示词方向

```text
inconsistent face, different outfit, extra fingers, broken hands, unreadable silhouette, cluttered background, random camera angle, bad anatomy, duplicated character
```

## 每日 10 分钟训练

1. 在 PixAI、Lexica 或 OpenArt 找 3 张你喜欢的图。
2. 不复制原提示词，先自己写“为什么有效”。
3. 把它改写成镜头提示词。
4. 加入连续性约束。
5. 生成或保存测试结果。
6. 把有效句子沉淀到 [[提示词素材库]]。

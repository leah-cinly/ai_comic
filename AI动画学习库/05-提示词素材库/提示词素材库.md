# 提示词素材库

这一页存放可复用提示词。来源可以是 PixAI、Lexica、OpenArt、Runway Academy、自己的实验结果。每条提示词最好记录用途和失败边界。

## 总公式

```text
[subject], [specific action], [shot size], [camera angle], [camera movement], [emotion], [lighting], [background function], [style], [continuity constraints]
```

中文理解：主体 + 具体动作 + 景别 + 机位 + 镜头运动 + 情绪 + 光线 + 背景功能 + 风格 + 连续性约束。

## 角色一致性

```text
same character, consistent facial features, consistent hairstyle, consistent outfit, consistent color palette, simple memorable silhouette, animation character design sheet
```

```text
character turnaround, front view, side view, back view, expression sheet, clean animation concept art, stable proportions
```

```text
keep the same character design from the reference, same hairstyle, same outfit, same age, same facial proportions, no costume change
```

## 动画镜头

```text
cinematic animated short film still, clear staging, readable silhouette, expressive pose, strong key light, background supports the character emotion, shot designed for animation
```

```text
pose-to-pose animation keyframe, anticipation pose, clear line of action, natural arc of motion, exaggerated but believable expression, follow-through in hair and clothing
```

## 分镜风格

```text
storyboard panel, grayscale, clear camera angle, readable action, animation timing notes, simple background, cinematic composition
```

```text
five-panel storyboard sequence, wide shot to close-up progression, clear eyeline match, action continuity, minimal line art
```

## 色彩脚本

```text
color script for animated short film, emotional lighting progression, limited palette, scene thumbnails, mood transition from calm to tension to release
```

## 镜头运动

```text
dolly in toward the character's face, emotional realization, smooth camera movement, shallow depth of field, animated film lighting
```

```text
tracking shot follows the character running through a narrow street, strong sense of speed, background motion blur, clear subject silhouette
```

```text
static camera, the character moves through the frame, strong silhouette, restrained motion, quiet emotional beat
```

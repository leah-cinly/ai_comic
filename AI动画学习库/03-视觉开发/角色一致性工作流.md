# 角色一致性工作流

## 目标

AI 动画短片最容易失败在“同一个角色每个镜头长得不一样”。角色一致性不是只靠一句 same character，而是靠角色表、参考图、服装规则、镜头约束和后期筛选共同完成。

## 角色设定表

必须包含：

- 正面、侧面、背面。
- 3 个表情：平静、害怕、开心/惊讶。
- 3 个动作：站立、奔跑、伸手。
- 服装固定规则：颜色、材质、配饰。
- 禁止变化：发型不能变、眼镜不能消失、鞋子颜色不能变。

## 生成顺序

1. 先生成角色概念图。
2. 选定 1 张主参考图。
3. 生成 turnaround。
4. 生成 expression sheet。
5. 生成 action sheet。
6. 再进入镜头图。

## 提示词模板

```text
same character, consistent facial features, consistent hairstyle, consistent outfit, simple memorable silhouette, animation character design sheet, front view, side view, back view, expression sheet, clean line art, stable proportions
```

## 镜头连续性约束

```text
keep the same character design from the reference, same hairstyle, same outfit, same color palette, same age, same facial proportions, no costume change
```

## 负面提示词

```text
inconsistent face, different hairstyle, different outfit, age change, extra accessories, missing glasses, random costume, duplicated character, bad hands
```

## 工具策略

- Midjourney / OpenArt / PixAI：适合快速找角色风格和表情。
- Stable Diffusion / ComfyUI：适合用 LoRA、IP-Adapter、ControlNet 做稳定角色。
- Photoshop / Krita：适合修正角色表，保留标准参考图。
- Runway / Pika / Luma / Kling：适合把已确定的关键帧转成视频。

## 检查表

- 角色轮廓是否一眼可识别？
- 发型和服装是否在每张图里一致？
- 表情是否服务故事，而不是随机可爱？
- 动作姿态是否有清楚剪影？
- 参考图是否少而明确？

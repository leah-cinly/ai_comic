# 场景与色彩脚本

## 场景不是背景

场景的任务是支持角色行动和情绪。短片里每个场景都要回答：角色为什么在这里，这个空间如何帮助或阻碍角色。

## 场景设定字段

| 字段 | 问题 |
| --- | --- |
| 地点 | 故事发生在哪里？ |
| 时间 | 白天、夜晚、雨天、黄昏？ |
| 功能 | 场景如何推动角色行动？ |
| 色彩 | 主色、辅助色、情绪色是什么？ |
| 光源 | 光从哪里来？是否能保持连续？ |
| 材质 | 木头、金属、玻璃、布料、雾气？ |

## 色彩脚本

色彩脚本不是配色图，而是情绪变化图。

示例：

| 段落 | 情绪 | 色彩 | 光线 |
| --- | --- | --- | --- |
| 开始 | 孤独 | 冷蓝、灰 | 弱光 |
| 发现 | 好奇 | 暖黄点光 | 局部发光 |
| 危机 | 紧张 | 高对比、暗红 | 斜向硬光 |
| 结尾 | 释放 | 暖橙、柔光 | 扩散光 |

## 提示词模板

```text
color script for animated short film, emotional lighting progression, limited palette, scene thumbnails, mood transition, consistent light direction, cinematic composition
```

## 场景生成模板

```text
animation background design, [location], [time of day], [weather], [story function], [limited color palette], [clear path for character action], [consistent key light direction]
```

## 练习

为同一个故事做 3 套色彩脚本：

- 温暖童话版。
- 冷色悬疑版。
- 高对比漫画版。

比较哪一套最适合角色变化。

# AI 动画短片制作知识库

这是一个 Obsidian vault，用来系统学习 AI 动画短片制作。重点不是只掌握某一个工具，而是建立“故事、镜头、表演、视觉一致性、AI 生成、剪辑复盘”的完整创作能力。

入口笔记：

```text
AI动画学习库/00-总览/AI动画学习总入口.md
```

## 常用命令

生成当天学习更新：

```powershell
.\scripts\update-learning-vault.ps1
```

同步到 GitHub：

```powershell
.\scripts\sync-to-github.ps1
```
